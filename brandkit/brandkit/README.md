# Hayati — Brand Kit v1.0 (July 2026)

Open `hayati-brand-guidelines.html` for the full guidelines (works offline).

## Contents
- `logo/` — mark + lockups. SVG (color / mono-sand / mono-night) and PNG (on-night, on-sand, Arabic lockup, 512 transparent mark). The lockup SVG uses live Rubik text; use PNGs where exact type matters.
- `tokens/` — `hayati-tokens.css` (custom properties) and `hayati-tokens.json` (structured tokens).

App icons and social/store graphics live in the sibling `branding-assets/` folder (separate download):
- `icons/` — iOS 1024, Play 512, Android adaptive fg/bg 432 (fg is transparent), discreet-mode alt icon.
- `social/` — Instagram post 1080×1080, Story 1080×1920, TikTok question card 1080×1920, Play feature graphic 1024×500 (each EN/TR/AR), X header 1500×500.

## Quick rules
- Dark-first: night `#231A33` backgrounds, sand `#F3E7D7` text.
- Gold `#D9A441` for premium/celebration only. Alert `#D96C5F` never in marketing.
- Rubik only (fallback Noto Sans / Noto Sans Arabic). Arabic body line-height 1.7.
- No faces or embracing figures on AR-market marketing surfaces.
- "Hayati" is a working title — clear trademark before public launch.
