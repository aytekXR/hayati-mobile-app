Comparison is exact (no tolerance comparator), which makes goldens engine-version-sensitive: expect intentional churn on a Flutter upgrade, re-baselined behind the explicit W4 golden-update flag. Mismatch diffs land in `failures/` next to the test (gitignored) and upload as CI artifacts.

## 4. Policies

Flaky test в†’ quarantined same day with an issue; >2 quarantined = next session objective is stabilization. Bug fix в†’ regression test in the same commit, named after the issue. Test speed budget: unit+widget suite <90s locally вЂ” protect it (fakes over emulators at unit level). LLM-dependent tests never call live models in CI (fixtures only); safety-path tests are the exception and run against the proxy's filter layer, which is deterministic. No test reads real user data, ever.
